# bs-issue-tracker
백석 이슈 트래커

## 기능정의
* [ ] 이슈 리스트
  * [ ] 분류 탭
    * [ ] Open 이슈 탭
    * [ ] Close 이슈 탭
  *[ ] 테이블 명세
    * [ ] 고유 ID
    * [ ] 제목 : 클릭하면 해당 이슈 상세로 이동
    * [ ] 작성자
    * [ ] 이슈 종류
    * [ ] 상태(status)

* [ ] 이슈 상세
  * [ ] 메타 정보
    * [ ] 제목
    * [ ] 작성자
    * [ ] 생성일
    * [ ] 수정일
    * [ ] 이슈 종류
    * [ ] 상태(status)
  * [ ] 이슈 내용 
  * [ ] 이슈 상태 변경 UI
    * [ ] 수정
    * [ ] 삭제 (영구적으로 데이터베이스에서 삭제)
    * [ ] 해결 진행 중
    * [ ] 해결 됨


* [ ] 이슈 등록 폼
  * [ ] 폼 입력 란
    * [ ] 제목
    * [ ] 내용
    * [ ] 등록하기(submit)
